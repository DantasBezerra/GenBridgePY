GenBridgePY

Servidor de acesso múltiplo à bancos de dados biológicos.

INTRODUÇÃO

Devido a grande demanda de ferramentas para análise de dados biológicos, principalmente os alunos encontravam um desafio! Aprender e manipular ttais ferramentas, mas, para minimiza a quantidade de ferramentas aprendidas, o GenBridgePY oferece um acesso múltiplo a tais dados.

DESENVOLVIMENTO

Atualmente existem várias maneiras de se desenvolver suas próprias soluções tecnológicas, uma das formas de realizar tal feito é fazendo uso da linguagem Pyhon, que oferece uma gama de possibilidades e bibliotecas, que nos permite desenvolver e aplicar soluções para cada necessidade.

APLICAÇÃO

Para atingir seu objetivo o GenBridgePY, foi construido com a linguagem Python e suas bibliotecas voltadas à bioinformática.
